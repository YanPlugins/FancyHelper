Citizens (NPC) 常用命令：
/npc create <name> [--type <type>] - 创建一个新 NPC
/npc remove - 删除当前选中的 NPC
/npc select <id> - 选中指定 ID 的 NPC
/npc rename <new_name> - 重命名 NPC
/npc type <type> - 更改 NPC 的实体类型（如 zombie, iron_golem）
/npc look - 使 NPC 始终看向玩家
/npc path - 设置 NPC 的行走路径
/npc equip - 开启装备编辑模式（右键给 NPC 穿装备）
/npc command add [-p] <command> - 给 NPC 添加点击执行的命令（-p 代表以玩家身份运行）
/npc skin <name> - 更改 NPC 的皮肤
/npc text - 进入对话编辑模式
/npc tphere - 将选中的 NPC 传送到你身边
注意：大部分命令需要先选中一个 NPC 才能执行。
